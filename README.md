## Como executar este projeto

Este projeto foi desenvolvido utilizando JavaScript. Para testar o código, certifique-se de ter o Node.js instalado e siga os passos abaixo:

1. Abra o terminal na pasta do projeto.
2. Execute o arquivo principal com o comando:
   `node index.js`
